# prenotazione-Teatro

Tentativo di creazione di una pagina web in grado di interagire con un DB con visualizzazione immediata dei risultati attraverso la visualizzazione di sedili di colore diverso. Cliccando sul sedile deve essere possibile visualizzare, solo per gli admin, le informazioni riguardanti a quello specifico sedile. 
Deve essere inoltre possibile creare, cancellare e modificare le varie prenotazioni. 
Gli admin devono poter avere accesso al DB in modo tale da poterlo stampare e comunque visualizzare completamente anche offline.
